# 业务流程图绘图规范

> 适用于：跨部门/跨角色业务流程、BPMN、泳道图（多角色协作场景）。
> 前置依赖：必须先读取 `common-rules.md` 通用规范。
> 模板目录：`examples/business-flow/`

---

## 一、节点间距规则

### 1.1 节点与泳道边界

节点任何边缘距离所属泳道（横向阶段或纵向角色）边线 ≥ 20px：

```
节点顶部 - 阶段顶部边线 ≥ 20px
阶段底部边线 - 节点底部 ≥ 20px
节点左侧 - 角色列左边线 ≥ 20px
角色列右边线 - 节点右侧 ≥ 20px
```

违反此规则会导致节点视觉上"贴在"或"骑在"泳道线上。

### 1.2 节点间距

- 同列垂直相邻节点：间距 ≥ 15px（节点底部到下一节点顶部）
- 同行水平相邻节点：间距 ≥ 30px
- 连线拐弯处与最近节点：间距 ≥ 10px

### 1.3 节点居中

节点应在其所属角色列内水平居中：
```
节点中心X = 角色列起始X + 角色列宽度 / 2
```

节点应在其所属阶段行内垂直居中（单节点时），或均匀分布（多节点时）。

## 二、泳道布局规则

### 2.1 尺寸填满

纵向角色泳道宽度之和必须精确等于外层容器宽度（减去左侧偏移）：
```
示例：容器宽度 1050，左偏移 30
30 + 200 + 200 + 200 + 200 + 220 = 1050 ✓
```

横向阶段泳道高度之和必须精确填满容器高度（减去顶部偏移）：
```
示例：容器高度 1000，顶偏移 60
60 + 160 + 120 + 120 + 120 + 120 + 140 + 160 = 1000 ✓
```

### 2.2 startSize 对齐

横向阶段泳道的 `startSize`（左侧标签宽度）必须等于纵向角色泳道的起始 X 偏移：
```
纵向泳道从 x=30 开始 → 横向泳道 startSize=30
```

### 2.3 左上角填充

横向阶段标签列与纵向角色标题行交汇处会产生空白角落，必须用同色矩形填充：
```xml
<mxCell id="xx" parent="容器" style="rounded=0;whiteSpace=wrap;html=1;fillColor=#cce5ff;strokeColor=#b0b8c0;fontSize=1;" value="" vertex="1">
  <mxGeometry x="0" y="30" width="30" height="30" as="geometry" />
</mxCell>
```
尺寸 = startSize × startSize（如 30×30），位置在容器标题下方、第一个纵向泳道左侧。

### 2.4 边框颜色

泳道边框使用浅色（`strokeColor=#b0b8c0`），避免与流程连线（黑色）视觉竞争。

## 三、节点父级规则

### 3.1 统一 parent

在泳道图中，所有业务节点和连线的 `parent` 必须指向同一个容器（通常是外层 swimlane）：
```xml
<!-- 正确：所有节点 parent="2"（外层容器） -->
<mxCell id="20" parent="2" ... />
<mxCell id="30" parent="2" ... />
<mxCell id="90" edge="1" parent="2" source="20" target="30" ... />
```

禁止将节点放入不同的子泳道 parent，否则跨泳道连线会路由失败。

### 3.2 泳道仅做视觉分区

纵向角色泳道和横向阶段泳道仅用于视觉分区，不作为节点的 parent。节点通过坐标定位到对应区域。

## 四、样式规则

### 4.1 外部/线下步骤

不在本系统内完成的步骤使用灰色斜体样式：
```
fillColor=#f5f5f5;strokeColor=#999999;fontSize=11;fontStyle=2
```

### 4.2 系统内步骤

本系统内的操作步骤使用蓝色样式：
```
fillColor=#dae8fc;strokeColor=#6c8ebf;fontSize=12
```

### 4.3 开始/结束节点

- 开始：`ellipse;fillColor=#fff2cc;strokeColor=#d6b656`
- 结束：`ellipse;fillColor=#d5e8d4;strokeColor=#82b366`

---

## 五、自检清单

生成 XML 后、渲染前，逐项检查：

- [ ] 所有节点距泳道边线 ≥ 20px
- [ ] 泳道宽度/高度精确填满容器
- [ ] 横向泳道 startSize = 纵向泳道起始 X
- [ ] 左上角填充块存在
- [ ] 所有节点和 edge 的 parent 一致
- [ ] 连线路径不穿过其他节点区域
