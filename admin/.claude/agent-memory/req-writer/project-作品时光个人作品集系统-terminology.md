# 项目术语表 — 作品时光个人作品集系统

> 本文件由 req-doc 流程在首次运行时初始化，后续各组 analyzer/writer 复用，保持术语一致。
> 更新日期：2026-06-28

## 核心实体术语

| 统一术语 | 含义 | 禁止混用 |
|---------|------|---------|
| 作品 | 系统核心实体，代表一件设计/开发作品 | 项目、案例 |
| 唯一标识 | 作品/分类的路径标识符，全局唯一 | slug、别名 |
| 原型版本 | 作品的某个迭代版本，如 V1/V2/V3 | 版本、迭代 |
| 原型图片 | 原型版本下的截图 | 截图、效果图 |
| 拆解文章 | 对作品的分析解读，富文本形式 | 文章、博客、项目拆解 |
| 分类 | 作品的归属类别，如 AI/医疗/外呼/CRM | 标签、类目 |
| 时间轴节点 | 成长轨迹中的一条年份记录 | 节点、里程碑 |
| 技能 | 关于我中的能力项，含名称与评分 | 能力、专长 |
| 个人介绍 | 关于我的简介与简历信息 | 简介、profile |
| 首页配置 | 展示站点首页的展示文案与精选作品设置 | 站点配置 |
| 访问量 | 作品详情被访问的累计次数（已去重） | 浏览量、PV |

## 操作术语（统一用法）

| 统一用法 | 禁止混用 |
|---------|---------|
| 新增 | 添加、创建 |
| 编辑 | 修改、更新 |
| 删除 | 移除、清除 |
| 发布 / 下架 | 上线/下线、开启/关闭 |
| 设为精选 / 取消精选 | 加精、置顶 |
| 启用 / 停用 | 正常/禁用 |

## 状态枚举

| 枚举类型 | 取值 | 说明 |
|---------|------|------|
| 作品发布状态 | 草稿 / 已发布 | 草稿不对访客公开 |
| 拆解文章发布状态 | 草稿 / 已发布 | 草稿不对访客公开 |
| 精选标记 | 是 / 否 | 是则用于首页精选展示 |

## 角色术语

| 统一术语 | 含义 |
|---------|------|
| 管理员 | 作品集主人，唯一具有后台维护权限的角色 |
| 访客 | 浏览展示站点的公众用户，无需登录 |

## 关键业务规则术语

| 术语 | 含义 |
|------|------|
| 草稿隔离 | 草稿状态的作品/拆解不对展示端公开，仅后台可见 |
| 级联删除 | 删除作品时一并删除其原型版本、原型图片、拆解文章 |
| 分类删除保护 | 分类被作品引用时阻止删除，需先迁移作品 |
| 访问量去重 | 同一来源短时间内重复访问作品详情只计一次 |
| 作品迁移 | 将作品从一个分类重新归属到另一个分类 |
