# 应用系统类项目可行性研究报告模板

> 本模板定义应用系统类项目可行性研究报告的完整文档结构，包含所有章节、内容要求和示例。
> feasibility-report 技能生成每个章节前必须对照本模板的内容要求和示例。

---

## 写作前必读：规范要求

生成任何内容前，确认符合以下要求：

- 每个章节的"内容要求"说明不出现在最终报告中，只用于指导生成
- 表格数据不足时填"待定"，不留空白单元格
- 项目类型不适用的章节标注"本项目不涉及，略"
- 数字金额统一用"万元"为单位，保留两位小数
- 时间节点用"T+N月"表示（T为可研报告批复时间）
