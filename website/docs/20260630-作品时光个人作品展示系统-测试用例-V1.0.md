## 三、引言

### 3.1 编写目的

本文档为作品时光个人作品展示系统的功能测试用例说明书，旨在为测试人员提供清晰的测试依据，确保系统各功能模块满足需求说明书要求，保障系统上线质量。

### 3.2 测试范围

本次测试覆盖以下端及功能模块：

**Web管理后台**：

- 登录模块：用户登录、注册、忘记密码
- 作品管理：作品列表、新增、编辑、删除、发布状态切换、精选标记切换
- 原型版本管理：版本列表、新增、编辑、删除
- 拆解文章管理：文章列表、新增、编辑、删除、发布状态切换
- 分类管理：分类列表、新增、编辑、删除
- 时间轴管理：时间轴列表、新增、编辑、删除
- 关于我管理：个人介绍编辑、头像上传
- 首页配置：首页数据配置
- 数据统计：访问量统计、数据分析
- 组织架构管理：部门管理、职位管理、用户管理
- 权限管理：菜单管理、角色管理

**Web展示端**：

- 首页：精选作品、能力图谱、最新思考、成长历程
- 作品列表：作品分类筛选、作品卡片展示
- 作品详情：作品内容展示、上下篇导航
- 时间轴：成长历程展示
- 关于我：个人介绍、联系方式
